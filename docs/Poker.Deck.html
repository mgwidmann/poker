Poker.Deck

  • Summary
  • Functions

Struct representing a Deck of cards used during gameplay. The deck keeps track of which cards are available to be drawn from and which have already been used. Additionally, it keeps track of the cards which are in player’s hands (though it doesn’t keep track of which players have which cards).

Creating a blank Deck struct like %Poker.Deck{} will by default have a :deck field which is filled in with the full card contents already shuffled (via the new call).
